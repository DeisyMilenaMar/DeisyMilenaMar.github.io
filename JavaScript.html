JavaScript

Variables var, let y const

var

Funcin:

  • var permite declarar una variable que puede ser re-declarada y reasignada.
1) Tiene mbito de funcin, lo que significa que es accesible en toda la funcin donde ha sido declarada.
2) Si se intenta acceder a una variable var antes de que haya sido declarada, devolver undefined debido a la elevacin (hoisting).
3) La variable var se declara en el contexto de ejecucin global o de funcin.
// var
var user = "Mary"; 
console.log(user); // Output: Mary
var user = "Joanna"; 
console.log(user); // Output: Joanna
user = "Mark"; 
console.log(user); // Output: Mark

let

Funcin:

  • let permite declarar una variable que no puede ser re-declarada en el mismo mbito.
1) Tiene mbito de bloque, por lo que solo est disponible dentro del bloque donde fue declarada.
2) Si se intenta acceder a una variable let antes de su declaracin, lanzar un ReferenceError.
3) Puede ser reasignada despus de la declaracin.
// let
let name = "Anna";
console.log(name); // Output: Anna
name = "Sara"; 
console.log(name); // Output: Sara
// let name = "John"; // Error: Identifier 'name' has already been declared

const

Funcin:

  • Las variables declaradas con const tienen mbito de bloque (block scope).
1) No pueden ser re-declaradas ni reasignadas.
2) El valor debe ser inicializado en el momento de la declaracin.
3) No se puede reasignar un nuevo valor, pero si es un objeto o arreglo, sus propiedades o elementos s pueden ser modificados.
// const
const age = 30;
console.log(age); // Output: 30
// age = 35; // Error: Assignment to constant variable.
const userInfo = { name: "Anna" };
userInfo.name = "Sara"; // Esto es permitido porque solo se modifica la propiedad
console.log(userInfo.name); // Output: Sara

Alcance (scope)

Global Scope

Funcin:

  • El alcance global (global scope) se refiere a cualquier variable o funcin declarada fuera de una funcin o bloque
1)  Las variables en el alcance global son accesibles desde cualquier parte del cdigo.
2)  Las variables declaradas fuera de una funcin o bloque tienen un alcance global.
3)  Son accesibles desde cualquier parte del cdigo.
var globalVariable = value;
let globalVariable = value;
const globalVariable = value;
Ejemplo
// Global Scope
let globalVariable = 'I am global';

function exampleFunction() {
    console.log(globalVariable); // Accessible desde dentro de la funcin
}

Local Scope (Function Scope)

Funcin:

  • El alcance local (local scope o function scope) se refiere a las variables declaradas dentro de una funcin.
1) Estas variables solo son accesibles dentro de la funcin en la que fueron creadas.
function exampleFunction() {
    let localVariable = 'I am local';
    let localVariable = value;
    const localVariable = value;
    console.log(localVariable); //  Accesible dentro de la funcin
}

console.log(localVariable); //  Error: No accesible fuera de la funcin

Block Scope

Funcin:

  • El alcance de bloque (block scope) se refiere a las variables declaradas con let o const dentro de un bloque {}.
1) Estas variables solo son accesibles dentro del bloque donde fueron declaradas.
2) Las variables declaradas con let o const estn limitadas al bloque donde fueron creadas (entre {}).
3) Esto incluye bloques if, for, while, etc
{
    let blockVariable = value;
    const blockVariable = value;
}
Ejemplo
{
    let blockVariable = 'I am in a block';
    console.log(blockVariable); //  Accesible dentro del bloque
}

console.log(blockVariable); //  Error: No accesible fuera del bloque

Scope Chain

Funcin:

  • La cadena de alcance (scope chain) permite que una funcin acceda a las variables definidas en un mbito superior (padre).
1) Si una variable no est definida en el mbito actual, JavaScript buscar en los mbitos superiores hasta encontrarla o llegar al mbito global.
let outerVariable = 'I am outer';

function outerFunction() {
    let innerVariable = 'I am inner';

    function innerFunction() {
        console.log(outerVariable); //  Busca en el mbito superior
        console.log(innerVariable); //  Busca en el mbito superior inmediato
    }

    innerFunction();
}

if-else statement

Funcin:

  • Evala mltiples condiciones secuencialmente.
1) Si la primera condicin es true, ejecuta el cdigo asociado a if.
2) Si la primera condicin es false, evala la condicin else if.
3) Si else if es true, ejecuta el cdigo asociado.
4) Si ninguna condicin es true, ejecuta el bloque else.
if (condition1) {
    // Code to execute if condition1 is true
} else if (condition2) {
    // Code to execute if condition1 is false and condition2 is true
} else {
    // Code to execute if both condition1 and condition2 are false
}

Componentes y atributos

  • (condition1): Expresin que evala la primera condicin y devuelve true o false.
  • { }: Bloque de cdigo que se ejecutar si la condicin es verdadera (true).
  • else if: Palabra clave que define una condicin adicional que se evaluar si la primera condicin es falsa.
  • (condition2) : Expresin que evala la segunda condicin y devuelve true o false.
  • { }: Bloque de cdigo que se ejecutar si la segunda condicin es verdadera (true).
Ejemplo
var place = "first";

if (place =="first") {
    console.log("gold medal");
} else if (place =="second") {
    console.log("silver medal");
} else if (place =="third") {
    console.log("bronze medal");
} else {
    console.log("no medal");
}

Loops

for loop

Funcin:

  • El for loop se utiliza para ejecutar un bloque de cdigo un nmero determinado de veces.
1)  Se usa cuando conoces exactamente cuntas veces deseas que se ejecute el cdigo.
for (initialization; condition; increment) {
    // code block to execute
}

Componentes y atributos

  • initialization: Se ejecuta solo una vez al inicio del bucle para establecer la variable de control.
  • condition: Evala si el bucle debe continuar ejecutndose.
  • increment/decrement:Modifica la variable de control en cada iteracin.
  • Bloque de cdigo: Se ejecuta en cada iteracin mientras la condicin sea true.
Ejemplo
for (let i = 1; i <= 5; i++) {
    console.log("Iteration:", i);
}

Componentes y atributos

  • Inicializacin (let i = 1): Se establece la variable i con valor 1.
  • Condicin (i <= 5):Mientras i sea menor o igual a 5, el bucle se ejecutar.
  • Incremento (i++): En cada iteracin, i aumenta en 1.

while loop

Funcin:

  • El while loop ejecuta un bloque de cdigo mientras una condicin sea true.
1)  Se usa cuando no sabes exactamente cuntas veces debe ejecutarse el cdigo, pero sabes que debe detenerse cuando una condicin especfica deje de ser verdadera.
initialization;
while (condition) {
    // code block to execute
    increment;
}

Componentes y atributos

  • initialization: Se ejecuta una vez antes de que el loop comience. Generalmente se usa para declarar e inicializar la variable de contador.
  • condition:Expresin que se evala antes de cada iteracin. Si es true, el loop contina; si es false, el loop termina.
  • increment: Se ejecuta al final de cada iteracin para actualizar el valor del contador.
  • // code block to execute: Bloque de cdigo que se ejecuta mientras la condicin sea true.
Ejemplo
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

do...while loop

Funcin:

  • El do...while loop ejecuta un bloque de cdigo al menos una vez, incluso si la condicin es false.
1)  Se usa cuando deseas que el cdigo se ejecute al menos una vez, independientemente de si la condicin es verdadera o falsa.
initialization;
do {
    // code block to execute
    increment;
} while (condition);
Ejemplo
let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Control de loops

// break
for (initialization; condition; increment) {
    if (expression) {
        break;
    }
}

// continue
for (initialization; condition; increment) {
    if (expression) {
        continue;
    }
}

Componentes y atributos

  • break: Termina el loop inmediatamente, independientemente de si la condicin sigue siendo true.
  • continue: Salta a la siguiente iteracin del loop sin ejecutar el resto del cdigo en el bloque.
Ejemplo
for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break;
    }
    console.log(i);
}

for (let i = 0; i < 10; i++) {
    if (i === 5) {
        continue;
    }
    console.log(i);
}

forEach()

    // code to execute on each element
});

Funcin:

  • Ejecuta una funcin proporcionada una vez por cada elemento del array en orden. No devuelve un valor, simplemente realiza una operacin en cada elemento del array.
1)  Se usa para iterar sobre los elementos de un array.
2)  Se emplea cuando deseas realizar una accin especfica sobre cada elemento del array, pero sin devolver un nuevo array.
3)  No crea un nuevo array; simplemente ejecuta una funcin sobre cada elemento.
const numbers = [1, 2, 3, 4];
numbers.forEach(function(number) {
    console.log(number * 2);
});
// Output: 2, 4, 6, 8

Modificar el atributo style

Funcin:

  • El mtodo setAttribute() permite establecer o modificar el atributo style de un elemento HTML usando una cadena de texto con propiedades CSS en lnea.
1) Reemplaza el valor completo del atributo style.
2) No es ideal cuando se desea mantener estilos existentes.
element.setAttribute('style', 'property: value; property: value;');
Ejemplo
<p id="myParagraph" style="color: red;">This is a paragraph</p>

<script>
  const paragraph = document.getElementById("myParagraph");
  paragraph.setAttribute("style", "color: blue; font-size: 18px;");
</script>

Establece todos los estilos en lnea

Funcin:

  • La propiedad cssText permite establecer todos los estilos en lnea mediante una nica cadena de texto CSS.
1) Similar a setAttribute() pero ms directa para trabajar con estilos en lnea.
2) Reemplaza todos los estilos en lnea previamente definidos.
3) cssText sobrescribe todos los estilos en lnea del elemento.
element.style.cssText = 'property: value; property: value;';
Ejemplo
<p id="myText">This is a text</p>

<script>
  const text = document.getElementById("myText");
  text.style.cssText = "color: green; font-weight: bold;";
</script>

Establece una propiedad CSS a style

Funcin:

  • El mtodo setProperty() Permite establecer una propiedad CSS especfica con una prioridad opcional (important), asegurando que el estilo se aplique incluso si existen otras reglas de estilo.
element.style.setProperty('property', 'value', 'priority');

Componentes y atributos

  • priority : (Opcional) asegura que esta regla tenga prioridad sobre otras reglas CSS.
  • setProperty(): define la propiedad CSS especificada.
Ejemplo
<p id="myParagraph">This is a paragraph</p>

<script>
  const paragraph = document.getElementById("myParagraph");
  paragraph.style.setProperty("color", "purple", "important");
</script>

Elimina una propiedad CSS especfica

Funcin:

  • El mtodo removeProperty() Permite eliminar una propiedad CSS especfica del atributo style de un elemento, restaurando el valor predeterminado o heredado.
element.style.removeProperty('property');
Ejemplo
<p id="myParagraph" style="color: blue; font-size: 18px;">This is a paragraph</p>

<script>
  const paragraph = document.getElementById("myParagraph");
  paragraph.style.removeProperty("font-size");
</script>

Componentes y atributos

  • removeProperty(): elimina la propiedad font-size de los estilos en lnea.
  • Si hay un estilo heredado desde CSS externo, este podra aplicarse despus de la eliminacin.

Eventos en el DOM

Funcin:

  • Permite asociar eventos a elementos HTML para responder a interacciones del usuario.
// Aadir evento a un elemento
element.addEventListener('eventName', callbackFunction);

// Eliminar evento de un elemento
element.removeEventListener('eventName', callbackFunction);

target.addEventListener(event, function);

Componentes y atributos

  • addEventListener() : Mtodo para registrar un evento en un elemento HTML.
  • event : Nombre del evento (por ejemplo, click, mouseover).
  • function : Funcin que se ejecutar cuando ocurra el evento.
  • target : Elemento HTML que recibe el evento (como document, window, button, etc.).
Ejemplo
<!-- aadir el evento al html  -->
<button id="my-button">Click Me!</button>

// Seleccionar el elemento objetivo
const target = document.getElementById('my-button');

// Definir la funcin para manejar el evento
function handleClick() {
    console.log('Clicked the body');
}

// Registrar el evento usando addEventListener
target.addEventListener('click', handleClick);

JSON a Objeto JavaScript y viceversa

Funcin:

  • JSON se utiliza principalmente para intercambiar datos entre un cliente y un servidor o entre diferentes partes de un sistema.
1) Se usa para serializar datos (convertir estructuras de datos en texto JSON) y deserializar datos (convertir texto JSON en estructuras de datos).
2) JSON tambin se utiliza para almacenar datos en archivos de configuracin, cach, o bases de datos.
3) JSON facilita la comunicacin entre sistemas heterogneos ya que es un formato independiente de lenguaje.
4) JSON se puede analizar y generar fcilmente mediante bibliotecas en todos los lenguajes de programacin modernos.

- Las claves deben estar entre comillas dobles ("").

  • Solo datos, sin mtodos ni funciones.

Convertir un JSON string a un objeto JavaScript 

const objectName = JSON.parse(jsonString);

Componentes y atributos

  • jsonString : String en formato JSON que se va a convertir a un objeto.
  • objectName: Nombre de la variable que almacenar el objeto resultante.
Ejemplo
const jsonString = '{"greeting": "hello"}'; // JSON vlido
const object = JSON.parse(jsonString);
console.log(object.greeting); // Output: hello

Convertir un objeto JavaScript a una cadena JSON 

const jsonString = JSON.stringify(objectName);

Componentes y atributos

  • stringify() : Mtodo que convierte un objeto JavaScript en una cadena JSON vlida.
  • jsonString: Nombre de la variable que almacenar el string JSON resultante.
Ejemplo
const data = { greeting: "hello" };
const jsonString = JSON.stringify(data);
console.log(jsonString); // Output: {"greeting":"hello"}
